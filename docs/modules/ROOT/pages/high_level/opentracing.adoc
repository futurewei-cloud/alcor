== Jaeger Integration

These creation workflow involves multiple microservice types:

. Jaeger client would report the metrics from each micro services which is managing a specific resource including VPC, subnet, port, route, IP and MAC then report the Jaeger Server
. All services are running inside the kubernetes cluster
